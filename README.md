# AVM Otopark Yönetim Sistemi
JavaScript tabanlı bu AVM otopark yönetim sistemi projesinde dinamik ücret hesaplama, DOM manipülasyonu ve kullanıcı etkileşimleri üzerine çalıştım. Sistem; araç türüne ve ek hizmetlere göre ücret hesaplama, işlem listeleme ve silme, gün sonu raporlama, ortalama ücret analizi ve localStorage ile tarayıcı tabanlı veri saklama özelliklerini içermektedir.
